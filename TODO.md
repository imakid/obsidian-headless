# Obsidian Headless 开发路线图

## 当前版本: v1.0.1

## 计划中功能

### 高优先级

- [ ] **语音操作支持** - 支持语音指令管理笔记
  - "创建笔记 今天想法"
  - "搜索内容 项目计划"
  - "今天日记 今天完成了..."

### 中优先级

- [ ] **笔记移动/重命名** - 支持移动和重命名笔记
- [ ] **笔记复制** - 快速复制笔记内容
- [ ] **批量操作** - 同时处理多个笔记

### 低优先级

- [ ] **标签管理** - 支持 Obsidian 标签操作
- [ ] **链接图谱** - 显示笔记链接关系
- [ ] **撤销功能** - 删除后支持恢复（trash）

## 已完成

- [x] 创建笔记（支持子目录）
- [x] 删除笔记（带确认）
- [x] 搜索笔记（标题、内容、模糊）
- [x] 查看笔记
- [x] 今天日记
- [x] 列出所有笔记和文件夹
- [x] 安全验证（路径遍历防护）
- [x] 多行内容支持
- [x] GitHub 发布
- [x] ClawHub 发布
